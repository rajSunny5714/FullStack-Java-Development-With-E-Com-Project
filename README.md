#FullStack Java E-Commerce Web Project

This project is a production-grade Full Stack eCommerce application built using Java, Spring Boot, Spring Security, JWT, JPA/Hibernate, MySQL, and React.js. The goal of this project is to implement real-world backend and frontend development practices while building a scalable and secure eCommerce platform.

Currently, I am learning and implementing concepts from a comprehensive Spring Boot Full Stack Developer course, where I am building this project step-by-step. The application includes features such as user authentication with JWT, product and category management, shopping cart functionality, order processing, and role-based access control.

On the frontend, React.js is used to create a responsive and interactive user interface, while the backend provides secure REST APIs using Spring Boot and Spring Security.

Through this project, I am gaining hands-on experience in full stack development, API design, database management, security implementation, and cloud deployment practices using AWS.
