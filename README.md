#Backend with Springboot and all...

Java, Spring Boot, Spring Security, JWT, JPA/Hibernate, MySQL, Docker. The goal of this project is to implement real-world backend practices while building a scalable and secure eCommerce platform.

Currently, I am learning and implementing concepts from a comprehensive Spring Boot Full Stack Developer course, where I am building this project step-by-step. The application includes features such as user authentication with JWT, product and category management, shopping cart functionality, order processing, and role-based access control.

Through this repo, I am gaining hands-on experience in fbackend development, API design, database management, security implementation, and cloud deployment practices using AWS.
